Module I - Analog optical processing, linear optical processing, spatial filtering using binary filters, inverse filtering, analog optical arithmetic, nonlinear optical processing, arithmetic operation

Module II - Recognition using analog optical systems, matched filter, joint transform correlation, amplitude modulated recognition filter, digital optical computing, nonlinear devices, integrated optical devices, SLM

Module III - Shadow casting and symbolic substitution, design algorithm of shadow casting system, polarization encoded optical shadow casting (POSC), POSC processing, symbolic substitution and optical implementation

Module IV - Optical matrix processor, linear algebraic processors, cellular logic architecture, programmable logic array- free space, fibre optic and acousto-optic techniques

Module V - Optical implementation of memory, holographic and wave guide optical interconnections
Optical Computing devices: Nonlinear Fabry-Perot etalon, optical transistor, threshold logic devices, devices using threshold logic gate components, optical computer


References:
1. Optical Computing- an introduction - MA Karim, AAS Awwal (Text)
2. Signal Processing using Optics - B G Boone
3. Optical Computing - D G Feitelson (Text)
